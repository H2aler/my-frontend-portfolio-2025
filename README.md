# 🚀 프론트엔드 포트폴리오 2025

## 📝 프로젝트 소개
이 프로젝트는 2025년에 제작된 프론트엔드 개발자 포트폴리오입니다. 최신 웹 기술과 트렌드를 반영하여 제작되었으며, 사용자 경험과 접근성을 최우선으로 고려했습니다.
